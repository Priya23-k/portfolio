## 📋 About The Project

This is my personal developer portfolio showcasing my skills, projects, and experience as a Full Stack Developer. The portfolio features a modern, responsive design with interactive elements, smooth animations, and a separate projects gallery page.

### ✨ Key Features

- 🎨 **Modern UI/UX Design** - Glassmorphism effects with animated gradients
- 📱 **Fully Responsive** - Optimized for all devices (Mobile, Tablet, Desktop)
- ⚡ **Smooth Animations** - Scroll-triggered animations and hover effects
- 🔍 **Projects Gallery** - Separate page with search and filter functionality
- 💬 **Contact Section** - Social links and email integration
- 🚀 **Fast Loading** - Optimized assets and performance
